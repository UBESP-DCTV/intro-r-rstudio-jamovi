Istruzioni

Negli spazi qui sotto puoi eseguire codice R a tua scelta Sperimenta pure a tuo piacere.

Una volta scritto del codice nella cella, premere il pulsante > Run Code per eseguirlo e vederne il risultato.

Prova a eseguire (anche modificando) una o più delle istruzioni riportate nelle varie sezioni (riportate a sinistra, o navigandole con i pulsalti Next Topic e Previous Topic). Esplorane altre a tuo piacimento.

Per ripristinare il contenuto originale di una cella, fare click sul pulsante Start Over della stessa.

Se servono suggerimenti su che comandi poter esplorare in ciascuna sezione, premere il pulsante Hint.

Facendo click su Start Over in fondo all’elenco delle sezioni a sinistra, tutto l’ambiente di lavoro verrà ripristinato e i progressi azzerati.
